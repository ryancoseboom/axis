# Vision

Axis is not a productivity application.

Axis is a personal reasoning engine.

Its purpose is to help people become someone they deeply respect by helping them spend today well.

The north star is:

> **Self-respect.**

Axis succeeds when it consistently creates a better ordinary Thursday than the user would have created alone.

Everything in this roadmap exists to support that goal.

---

# Guiding Principles

Axis is built around a few immutable principles.

- Today is computed, not stored.
- Reasoning is deterministic.
- The app should disappear while the user is living.
- Simplicity is preferred over capability.
- Identity is more important than productivity.
- Development matters more than completion.

# Version One Definition

Axis Version One is complete when:

- Identity meaningfully influences Today.
- Domain Models can be generated for new pillars.
- Knowledge Maps evolve through practice.
- Programs guide structured development.
- The Decision Graph consistently produces thoughtful recommendations.
- Morning, Now, and Evening form one coherent daily experience.
- Axis passes the Thursday Test for ordinary life.

---

# North Star

Axis is not trying to help people get more done.

Axis exists to help people spend today in a way that their future self will respect.

Everything else is in service of that goal.
