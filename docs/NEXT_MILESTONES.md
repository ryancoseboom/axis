# NEXT_MILESTONES.md

## Current Objective

Transform Axis from a UI-driven prototype into a reasoning engine whose
daily plan emerges from a Decision Graph.

## Milestone 1 (Current)

Implement the first Decision Graph pipeline.

**Definition of Done** - Inputs are converted into a Decision Graph. -
Candidate Decisions are generated. - A deterministic selector produces
Today. - Every Today output is traceable to graph nodes. - Confidence is
explainable. - Existing UI continues to work with minimal visual
changes.
